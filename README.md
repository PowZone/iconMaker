Image Resizer for iOS
==============

iOS 개발자를 위한 아이콘 메이커 어플리케이션 입니다.

`icon.png` 를 넣으면 `29pt.png, 29pt@2x.png, 29pt@3x.png, 40pt.png`등등 iOS 앱에 필요한 아이콘 사이즈를 생성합니다.
